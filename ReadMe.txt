Пример текстового файла "input.txt":

1. Какая утилита используется обычно для сборки майнлайн ядра Linux?
-Cmake
+Make
-Gradle
-Maven
-Basel

После выполнения программы будет создан файл "output.txt" с результатами преобразования в формат GIFT:

1. Какая утилита используется обычно для сборки майнлайн ядра Linux? {
~Cmake
=Make
~Gradle
~Maven
~Basel
}

Обратите внимание, что вариант правильного ответа помечен символом "=", а варианты неправильных ответов - символом "~".
