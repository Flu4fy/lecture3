Пример текстового файла "input.txt":

Как называется столица Франции?
Париж
Москва
Лондон

Кто написал произведение "Война и мир"?
Лев Толстой
Федор Достоевский
Антон Чехов

Какие из перечисленных фруктов являются цитрусовыми?
Апельсин
Яблоко
Груша
Лимон

После выполнения программы будет создан файл "output.txt" с результатами преобразования в формат GIFT:

::Как называется столица Франции?{
~ Париж
~ Москва
~ Лондон
}

::Кто написал произведение "Война и мир"?{
= Лев Толстой
~ Федор Достоевский
~ Антон Чехов
}

::Какие из перечисленных фруктов являются цитрусовыми?{
= Апельсин
~ Яблоко
~ Груша
= Лимон
}

Обратите внимание, что вариант правильного ответа помечен символом "=", а варианты неправильных ответов - символом "~".
